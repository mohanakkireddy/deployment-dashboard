1
Create GitHub OAuth App

Go to GitHub Developer Settings → OAuth Apps → New OAuth App

  • Application name: Deployment Dashboard
  • Homepage URL: Your GitHub Pages URL
  • Authorization callback URL: Same as Homepage URL
2
Deploy Token Exchange Proxy

Since GitHub Pages is static, you need a small backend to exchange the OAuth code for a token.

Options:

  • Cloudflare Worker (Free) - See code below
  • Netlify Function
  • Vercel Serverless Function
3
Update Configuration

Edit src/app/core/services/oauth.service.ts and set:

  • clientId: Your OAuth App's Client ID
  • tokenProxyUrl: Your proxy endpoint URL
  • redirectUri: Your GitHub Pages URL
